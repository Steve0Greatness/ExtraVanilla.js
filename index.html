Add Element

document.addElement
This function has 3 parameters. The first one you need to set to the tagname of the element that you'd like to make. The second should be set to an the element that you'd like it to be a child of, set it to something like document.getElementById("foo"). In the final parameter, you need to set it to an array of the attrobutes; they should all be layed out something like { name: "foo", value: "bar" }; I'd also just like to mention 3 things that are special in here:
Attrobute Usage
style another array of objects, it defines the style attrobute, layed out like { name: "color", value: "#f00" }.
text innerText
html innerHTML
class Either an array or a string of all classes you'd like to use.

  1. Body*: Body of notifacation
  2. Title: Self explanatory
  3. Icon: Icon sent
  4. Link: link opened when the notifacation is clicked.
  5. Vibration: Should the device vibrate when this is sent?
